
# Introduction

There are currently 16 built-in hooks in React.

1. State Hooks

    1. useState
    2. useReducer

2. Context Hooks

    1. useContext

3. Ref Hooks

    1. useRef
    2. useImperativeHandle

4. Effect Hooks

    1. useEffect
    2. useLayoutEffect
    3. useInsertionEffect

5. Performance Hooks

    1. useMemo
    2. useCallback
    3. useTransition
    4. useDeferredValue

6. Resource Hooks

    1. use

7. Other Hooks

    1. useDebugValue
    2. useId
    3. useSyncExternalStore
