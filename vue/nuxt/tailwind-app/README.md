# Installation

```shell
npx nuxi@latest init tailwind-app
cd tailwind-app
npm install
npm run dev
```

# Intall tailwind

```shell
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
```

```shell
mkdir assets
mkdir css
touch main.css
```
